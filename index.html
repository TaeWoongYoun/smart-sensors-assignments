About Project

LEGO Mindstorms EV3를 활용한 자율주행 로봇 제어 시스템입니다. 터치·컬러·자이로·초음파 4종 센서를 통합하여 색상 인식 기반 경로 추적, 장애물 회피, 지시색에 따른 복합 주행 패턴을 구현했습니다. 시작·종료 지점 인식과 주행 시간 측정, 안전 종료 조건까지 포함합니다.

센서 통합 제어

  • 터치 센서 · 주행 시작 트리거
  • 컬러 센서 · 경로 색상 인식 및 위치 판단
  • 자이로 센서 · 정밀 회전 각도 제어
  • 초음파 센서 · 장애물 거리 측정 및 회피

주행 동작

상황동작
시작/종료 지점특정 색(검정·초록·노랑·갈색) 감지 시 LED 제어 및 주행 시작/종료
검정 구간통과 횟수 실시간 카운팅 (10회 진입 시 자동 정지)
주행 시간시작~종료 시간을 초 단위로 측정·디스플레이 출력
파랑색 감지통과한 검정 구간 개수만큼 경고음(beep) 반복
빨강색 감지½ 감속 → 반시계 90° 턴 → (검정구간×10cm) 전진 → 복귀 → 재가속
장애물 50cm 미만경고음 + ½ 감속하여 접근
장애물 10cm 미만20cm 후진 → 시계 90° 턴 → (검정구간×10cm) 우회 → 복귀 → 재가속
